GuiaDevJr Bot

Deixe a sua comunidade assistida de um jeito fácil

O que ele faz por você?

1. Cria um menu com comandoas para ajudar/auxiliar com o dia a dia de Dev Jr

2. Possui uma variedades de opções para links, textos de ferramentas e consultas simples.

3. Sugestões de músicas, podcasts, para pessoas desenvolvedoras se atualizar enquanto se diverte.


Comandos:



/code - Visual Code versão online!


/doc_tech - Acesse a documentação da tecnologia que quiser


/git - Responde comandos mais usado do GIT bash!


/ping - Responde 'Pong!'


/playlist_azureacademy - Ouça a melhor playlist de estudos, segundo Azure Acadmey (Rubens) - Rock!


/playlist_ballerini - Ouça a melhor playlist de estudos, segundo Rafaella Ballerini - Rock!


/podecast_news_dia - Ouça a melhor PodCast de notícias de tecnologia diariamente, segundo Robson Mendonça


/podecast_news_sabado - Ouça a melhor PodCast de notícias de tecnologia aos sábados, segundo Robson Mendonça


/toolbox - Criado em 2012, o 4Devs é um portal com ferramentas online e gratuitas para programadores.
